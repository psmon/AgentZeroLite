### Per-feature deep-dive (M0009 surface specific)

| Surface | Concern | Verdict |
|---|---|---|
| `TokenUsageCollector` filesystem reads | Reads `~/.claude*/projects/**/*.jsonl` and `~/.codex/sessions/**/*.jsonl`. Read-only via `FileShare.ReadWrite \| Delete`. No symlink-following beyond .NET defaults. Path glob is anchored to `Environment.GetFolderPath(SpecialFolder.UserProfile)` — no untrusted prefix. | **Clean.** |
| Stored data | `TokenUsageRecord` columns are token counts + sessionId + cwd + model + raw requestId. No auth tokens, no transcript content. `AccountKey` = orgUuid or profile-dir name — not secrets. `.credentials.json` is *read* (orgUuid only) but never persisted to our DB. | **Clean.** |
| Bridge ops | `tokens.aliases.set` accepts user-controlled `vendor`, `accountKey`, `alias` strings and EF-parameterized UPSERT. No SQL injection risk. **Low** — string lengths unbounded; a misbehaving plugin could write very long `alias` rows. Mitigated by `maxLength=60` on the JS side input box, but server side has no enforcement. Filed for follow-up; not release-blocking. | **Low.** |
| `OfficialPluginCatalog` HTTP | Hardcoded `owner=psmon`, `repo=AgentZeroLite`, `branch=main`. User cannot redirect traffic. UA string `AgentZeroLite-OfficialCatalog/1.0`. 20s timeout. Network failures degrade gracefully (empty list + log). | **Clean.** |
| Plugin install URL plumbing | `OnInstallOfficial` populates `GitUrl` with the catalog's `GitFolderUrl` (also hardcoded prefix). The downstream `WebDevPluginInstaller.InstallFromGitUrlAsync` already validates manifest, manifest.id regex, file count limit (200), per-file size cap (25 MB), and zip-slip — none of those guards are bypassed. | **Clean.** |
| Dashboard rendering | All user-controlled strings (orgUuid, alias, cwd, sessionId, model) routed through JS `escape()` (HTML-entity encoding `& < > "`) before innerHTML. **Info** — `tooltip` (`title=`) attributes also escaped, including `r.cwd`, which could legitimately contain newlines if a malicious transcript line wrote them. Newlines in title attribute degrade visually, no XSS. Acceptable. | **Info.** |
| Reset action | `tokens.reset` wipes only `TokenUsageRecords` + `TokenSourceCheckpoints` via EF `ExecuteDelete`. Other DbSets untouched. UI confirms via `window.confirm` before invoking. | **Clean.** |
| WPF dialog code path | `InstallPluginPickerDialog` async-loads catalog on `Loaded`; combo selection populates `tbOfficialUrl` (display-only, not used by installer). Install button passes the catalog-supplied URL into the existing installer pipeline. | **Clean.** |

## Findings list

- **L1 (Low)** — `TokenAccountAlias.Alias` and `AccountKey` strings have no
  server-side length cap. A long alias would just look ugly in the table;
  no security impact, but a tidy `if (alias.Length > 200) throw` would be
  belt-and-suspenders. Filed as future polish; **not a release gate**.
- **I1 (Info)** — Cwd values surfaced in tooltips inherit whatever the CLI
  wrote into the transcript; newlines visually break titles but cannot
  escape to script execution because of consistent `escape()`. Documented
  for awareness.
