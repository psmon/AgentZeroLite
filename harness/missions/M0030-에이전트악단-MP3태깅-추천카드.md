# 요청 (Brief)

M0029 로컬 MP3 플레이리스트의 2차 개선.

1. **리얼타임 분류 재조정** — 사전분류(LLM)보다 우선순위로 분류되었어도
   재생 중 음향 판정으로 재조정 가능해야 함.
2. **음악적 느낌 태깅** — 음향에 분류되는 음악 느낌 태깅이 많음(AST 무드 라벨).
   재생 중 추가 태그로 업데이트. 태깅은 **장르 / 가수 / 음악적느낌 / 악기**로
   구분될 수 있어야 함.
3. **무대 규칙 재정의**:
   - 솔로 판정 시 1인 가수 보컬만 음악에 맞게 등장, **가수는 사라지면 안 됨**.
   - 노래(남성/여성) 음향 판정 시 캐릭터 노래 애니 액션, 일반일 때 기존(idle) 애니.
   - **그룹 판별 시에만** 걸그룹 모드 전환 — 걸그룹은 **신나는 음악 + 그룹 판정**이
     함께되어야 함. 신나는 음악이어도 1인 여성 솔로면 1인 모드.
4. **검색 강화** — 가수 검색 + 1차 필터로 장르 선택 시 하위 포함된 가수 태그가
   나와 원클릭 필터 가능.
5. **느낌 카드 자동추천 모드** — 선택적 플레이리스트(태깅 필터 검색 기반)와 별도로,
   음악적 느낌 카드를 클릭하면 재생되는 자동추천 모드. 카드는 LLM이 태그를 파악해
   자동생성(포함된 가수·느낌 간략 설명), "카드생성하기"로 추가되며 **저장**되어
   음악 검색 필터 요소를 가짐.
