# 요청 (Brief)

AgentZero 오리진으로부터 OS제어 기능을 가지고 오려고합니다.

대상위치 : D:\Code\AI\AgentWin\Project : 형제프로젝트로 하위참고

이 프로젝트의 구조에 맞게 가지고와야 하며 다음 목표를 달성

OS를 제어하는
CLI : CLI기능에서 윈도우를 효율적으로 제어하는 방법을 흡수
LLM TOOL 체인펑션콜 : 평션콜에서 윈도우를 효율적으로 제어하는 방법을 흡수

문서업데이트 :
- 작업 완료후 CLI설명문서및 LLM 툴체인 설명등의 문서업데이트(하네스뷰 문서포함)
- Readme OS 자동화 설명업데이트

테스트 :
이 기능을 가지고 오고나면 해당 기능을 참고 E2E테스트가 가능합니다.
이 기능을 도입후 CLI의 OS제어기능을 이용해 이 프로그램을 실행하고 기능을 파악합니다.
(기능접근만 확인.. 기능수행x)
이 CLI에 의해 e2e 테스트수행시 스크린샷또는 로그캡처등은 tmp/os-cli 하위에 남깁니다.(관련 하네스 업데이트)
