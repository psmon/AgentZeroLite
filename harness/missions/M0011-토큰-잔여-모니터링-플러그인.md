# 요청 (Brief)

Project/Plugins 하위에 공식 플러그인 기능을 하나더 추가하려합니다.

기능 : 클로드 코드 잔여 토큰 모니터링
획득방법 : claude-hud참고 이 플러그인이 기록한 최신값을 가지고옴

획득사이클
-최초에 한번 최근데이터 
-설정에 자동Sync가 있고 분설정가능 : 기본은 On으로 1분단위 갱신
-최신값 체크가 중요하기에 별도저장할필요없음

다음 콘솔에서 작동하는 모니터링을 참고
미니멀한 UI버전으로(위젯느낌으로)
설정은 멀티 계정병 On/Off를 할수 있으며 On된것만 보여준다.
```
[Opus 4.7 (1M context)]                                                                                                                                                                          
AgentZeroLite git:(main)                                                                                                                                                                          
Context █████░░░░░ 53%                                                                                                                                                                            
Usage ██░░░░░░░░ 19% (resets in 17m)                                                                                                                                                              
Weekly █░░░░░░░░░ 9% (resets in 6d)  
```
타크테마와 화이트테마 두개를 지원 
HUB UI모드도 있는데 순수 텍스트모드도 설정지원(아스키없이 순수 텍스트정보모드, 텍스트일때 텍스트색상 배경색상 설정가능 배경은 투명도 조정가능)

사전조사:
- 최근 전체 모니터링 데시브도가 구현되었으며 계정별 구분하는방법을 이해할것
- claude-hud 플러그인을 조사하면 잔여 토큰 Usage(일사용), Weekly(주사용) 잔여토큰을 확인할수 있습니다. 해당 플러그인이 참조하는 정보를 통해 획득가능할것으로 예상(중요)


구현하기전 펜슬 디자인 리뷰먼저
