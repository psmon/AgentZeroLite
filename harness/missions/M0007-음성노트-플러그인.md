# 요청 (Brief)

WEB DEV에는 최근 개발한 웹 Wasm을 이용한 기능이 있으며 플러그인 설치가 가능합니다.

이제 보이스 음성 노트 플러그인을 분리되어 작성하려고합니다.
플러그인은 zip형태로 설치하는 방법과 이번에 git url을 통해 설치되는 두가지방식

추가플러그인 핵심기능
- STT를 이용해 음성노트를 작성
- 음성노트의 목록관리가 가능하며 new 하면 새로운 음성노트
- 무조건 전체녹음이 아닌 음성감지기능(기존구현됨)을 활용해 인식된 음성을 텍스트화 진행
- 음성노트 : 음성녹음 일시정지기능이 있습니다. 인식률체크를 위한 민감도 설정기능이 있습니다.
- 요약하기 : 요약하기를 누르면 녹음된 텍스트를 분석해 요약저장을 합니다.( 분석시 Max토큰 오류시 1/2분할 정크처리 요약  )
- 노트저장계층
  - 순수하게 저장된 시간대별 텍스트 로우데이터가 있습니다.
  - 요약된 노트텍스트가 있습니다.
  - 노트작성에 이용된 모델및 토큰소요정보등 노트에 관한 추가 메타 정보가 있습니다.

Agent 공식 플로그인 생성경로
- Project/Plugins : 하위에 프러그인별 생성
- Project/Plugins/README.md : 플러그인들을 소개하는 페이지(설치방법)

git주소 설치 FLOW
- manifest를 확인해 다운받는 파일 목록을 확보합니다.
- 유효한 플러그인이인지 유효검사
- 이상없을시 설치진행 , 설치완료후에는 메뉴

추가고려
- 이 플러그인을 개발하기위해 WASM 네이티브 이용 인터페이스가 없거나 구현체가 없을수 있습니다. 없는경우 확장합니다.
- Project/Plugins 하위 플러그인 프로젝트는 외부에서 설치가능한 분리된 프로젝트입니다. ( 컴파일시 포함x)
- 이 프로젝트의 github 접근주소 : https://github.com/psmon/AgentZeroLite
