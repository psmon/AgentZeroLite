# 요청 (Brief)

음성인식을 개선하려고함.. 위스퍼넷 미디엄 모델이용
지금 흘러나오는 소리로 인식 ( 강의중 )
(개인이 가까울때는 인식완료.. 환경음이 있을때 소리가 적을때 인식개선)
감도설정관련 유닛테스트를 이용해 먼저 테스트 진행

연관 관련 옵션을 엄겸한(개인)-현재기준 / 느슨함(환경음 고료) 구분하고자함
현재수준은 엄격함이니, 느슨함을 옵션을 추가


개인클리어목소리 + 환경음이 있는 상태 인식등의 감도조정이 가능하도록 후속 개선은 후진행예정

- [ ] `VoiceSensitivityProfile` enum (Strict / Loose) 도입 — Strict가 현재 동작과 동일
