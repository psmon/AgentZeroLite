# 요청 (Brief)

스킬이용 : harness-kakashi-creator

AgentBot 의 AI모드에서 다음과 같이 시각장애인을 위한 보이스 모드를 제공하려고합니다.
AgentBot에 음성모드일때 위임모드를 추가해 ClaudeCLI에 요청을 하고 응답을 전달합니다.

"에이전트 위임" 하면 상태가 변경되어 해당모드로 작동됨
"에이전트 위임중단" 하면 다시 AIMode가 응답하는 원래 모드로 복귀함

위임모드일시 활성화된 claude중 하나와 요청을 하고 응답결과를 요약해 전달합니다.

"쿠팡에서 40대가 좋아하는 물건 찾아죠" 할시...
AgentBot-> 클로드에게 요청
클로드가 수행한 응답값을 -> AgetBot에 전달
음성출력 : 클로드가 응답한 값을 깔끔하게 음성으로 출력

## 구현세부지침
AgentLoop 에서 펑션콜을 이용해 처리하고 상태값에따라 행위가 변경됨으로 액터 상태머신을 잘활용해 대응

## 사전조건체크
- claude 라는 터미널과 대화를 하기위해 활성화가 되어야함
- 활성화 안된경우 claude 설정 cli를 구동함
- 웹탐색을 위해 playwrite 를 claude 에게 요청해 가능한지 확인해야함
- 설치없을시 설치를 요청해 자동진행하고 결과를 알려줌
  (설치중 클로드의 허용요청이 있는경우도 음성으로 진행해 클로드코드가 설치할수있도록 유도 엔터키를 전송해줘야함)
- 클로드CLI는 인증된상태여야하며 최초 인증까지 도와주는것은 PASS

## Acceptance
- [ ] 음성으로 "에이전트 위임" 발화 → AgentBot 상태가 위임 모드로 전환
- [ ] 위임 모드에서 사용자 발화는 강제로 Claude 터미널 relay 경로(Mode 2)로 흐름
- [ ] Claude의 응답이 `done` 메시지로 요약되어 음성으로 출력 (기존 TTS 경로 재사용)
- [ ] "에이전트 위임중단" 으로 일반 AI 모드 복귀, KV 캐시 리셋
- [ ] Claude 터미널이 없을 때는 음성으로 안내
- [ ] VoiceCommandInterceptor 단위 테스트 통과
