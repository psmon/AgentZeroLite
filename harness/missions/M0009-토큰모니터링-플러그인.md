# 요청 (Brief)

최근 플러그인 기능이 완성되었으며
Project/Plugins 하위에 공식 플러그인 기능을 하나더 추가하려합니다.

기능 : 토큰 모니터링을 조사
작동형식 : 클로드 코드및 코덱스가 토큰 기록하는 공간을

획득방식 :
AgentZero Lite 구동기준 해당경로를 파악후 텔레메트리를 파악  벤더별 계정별로 추적해야함 
텔레메트리 DB를 만들고 1분마다 토큰 사용량을 추적하여 저장합니다.(텔레메트리기반)
그리고 텔레메트리 기능을 제공합니다.

플러그인의 웹은 AgentZero가 가진 텔레메트리를 대시보드화해 보여줄수 있습니다.
웹은 수집방법및 제어에대해 관여하지 않고 대시보드 기능을 가집니다.
(벤더별/계정별 구분되어 표시)

사전조사:
- 클로드 코드CLI, 코덱스CLI등이 토큰사용을 어디위치에 조사하는지 알아야함
